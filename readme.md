## To run the application
### Create a virtual environment in WSL
```python3 -m venv .env```

## Activate the environment
``` source .env/bin/activate```

## Install the packages
```pip install -r requirements.txt```

## run the app.py
```python3 app.py```

## copy the path in your browser

```http://127.0.0.1:8080```
